AI 조율 방식

코드를 대신 쓰는 AI가 아니라, 역할을 나눈 AI 팀으로 검증합니다.

핵심은 "AI로 만들었다"가 아니라 어떤 문제를 어떤 역할의 AI에게 맡기고, 결과를 어떻게 확인해 업무 성과로 연결했는지입니다.

저는 개발 직무 출신이 아닙니다. 그래서 더 명확하게 문제를 쪼개고, AI에게 맡길 역할과 사람이 직접 검증할 기준을 먼저 정합니다.

각 사례는 문제 정의, AI 역할 분담, 산출물 검증, 실제 업무 적용의 순서로 남겼습니다. 이 페이지는 그 과정을 직무 포트폴리오의 증거로 연결하는 공개 검증 surface입니다.

01

문제 정의

반복 행정, 누락, 품질 편차처럼 실제 업무에서 아픈 지점을 먼저 고정하고 자동화 가능한 단위로 나눕니다.

02

역할 분담

Claude, ChatGPT, 자동화 도구를 분석가, 구현자, 리뷰어, 문서화 담당처럼 나눠 한 모델에 의존하지 않습니다.

03

검증 루프

출력물을 그대로 믿지 않고 샘플 데이터, 누락 케이스, 화면 동작, 운영 흐름을 확인하며 반복 수정합니다.

04

업무 연결

데모로 끝내지 않고 실제 채용, 서명 수집, 화환 접수, 직무 분석 업무의 시간과 오류를 줄이는 방향으로 연결합니다.

검증 사례

직무 포트폴리오의 DX 사례를 4개 증거 블록으로 통합했습니다.

각 블록은 문제, AI 조율 과정, 데모 영상, 정량 성과를 한 자리에서 보여줍니다.

HR · Automation직무 포트폴리오 dx1

경조화환 주문 시스템 자동화

문제

카톡·문자로 건별 접수하던 경조사 신청은 누락, 전달 지연, 회계 증빙 오류가 생기기 쉬웠습니다.

AI 조율 과정

업무 흐름을 신청자·담당자·업체·회계로 분해하고, AI에게 로그인/폼/알림/시트 적재 로직을 역할별로 설계·검토하게 했습니다.

증명

재현 데모는 보안 로그인, 신청 입력, 자동 알림, 접수 데이터 적재 흐름을 보여줍니다.

성과

휴먼에러 0건, 반복 행정 약 90% 절감.

HR · E-Signature직무 포트폴리오 dx2

전자 서명 수집 프로그램

문제

생산직 550명의 서약서 서명을 종이와 PC 중심으로 받으면 기간이 길고 미제출자 추적이 어렵습니다.

AI 조율 과정

AI에게 모바일 입력 UX, Canvas 서명 저장, Google Sheets 적재, 미제출자 모니터링 흐름을 나눠 설계하게 하고 실제 현장 제약으로 검증했습니다.

증명

재현 데모는 휴대폰 서명 입력, 이미지 변환, 제출 데이터 확인 흐름을 보여줍니다.

성과

550명 전원 수합, 약 2주 작업을 3일로 단축.

직무 키워드 분석 에이전트

문제

인터뷰와 설문 텍스트를 사람이 읽어 직무별 역량으로 정리하면 기준이 흔들리고 시간이 많이 걸립니다.

AI 조율 과정

AI를 직무 분석가 페르소나로 고정하고, 분류 기준·출력 포맷·검수 질문을 반복해 결과 편차를 줄였습니다.

증명

재현 데모는 텍스트 입력에서 키워드 추출, 직무별 정리, 분석 결과 표준화까지의 흐름을 보여줍니다.

성과

정성 데이터의 직무별 역량 정량화, 외부 지원자 EVP 600건+ 분석으로 확장.

채용공고 자동 생성 도구

문제

AI 조율 과정

AI에게 채용 데이터 구조화, 포스터 문안 생성, 화면 구현, 결과 검수 역할을 나눠 맡기고 배포 가능한 공개 도구로 묶었습니다.

증명

성과

Google Sheets 또는 AI 생성 채용 데이터에서 포스터 이미지를 자동 생성하는 공개 배포 프로젝트.

자주 묻는 질문.

정말 비개발자인가요?

네, 개발 직무 출신이 아닙니다. 대신 AI 역할 분담, 검증 기준, 업무 적용 흐름을 직접 설계해 실제 작동하는 결과로 연결합니다.

이 페이지의 목적은 무엇인가요?

직무 포트폴리오에 적힌 DX 사례가 실제 작동했음을 확인할 수 있도록 과정, 방법론, 데모, 성과를 한 페이지에 모은 검증 surface입니다.

가장 중요한 증거는 무엇인가요?

데모 영상은 보조 증거입니다. 핵심은 문제를 어떻게 정의했고, 어떤 AI를 어떤 역할로 조율했으며, 결과를 어떻게 검증했는지입니다.

검증하는 사례는 무엇인가요?

경조화환 주문 자동화, 전자 서명 수집, 직무 키워드 분석, 채용공고 자동 생성 네 가지 HR DX 사례입니다.

직무 포트폴리오와는 어떤 관계인가요?

직무 포트폴리오는 맥락과 역량을 설명하고, 이 페이지는 그 안의 DX 사례가 실제 작동했다는 증거를 제공합니다.

연락은 어떻게 하나요?

GitHub Issue로 프로젝트 검증, 데모 확인, 협업 제안을 보낼 수 있습니다.
