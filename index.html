FAQ

이 사이트는 어떤 포트폴리오인가요?

Lucas Sung의 Vibe Coding / GitHub-connected portfolio입니다. AI 에이전트와 함께 만든 프로젝트를 GitHub 저장소, live demo, README, spec, Knowledge OS evidence로 검증 가능하게 보여줍니다.

이 사이트는 경력 포트폴리오인가요?

아닙니다. 이 페이지는 작성형 커리어 포트폴리오가 아니라 GitHub evidence portfolio입니다. 경력 포트폴리오로 연결되기 전, 실제 코드와 배포 증거를 먼저 제시하는 표면입니다.

대표 프로젝트는 무엇인가요?

대표 프로젝트는 hr-documents, nerdspecs, hermes-ops-dashboard-page, Video Knowledge Ingestion Pipeline입니다. 각각 HR 운영 자동화, AI 프로젝트 문서화, AI agent operations dashboard, Knowledge OS evidence ingestion을 증명합니다.

AEO/GEO를 위해 어떤 구조를 갖추나요?

명확한 프로젝트 taxonomy, FAQ, JSON-LD, llms.txt, canonical URL, Open Graph/Twitter metadata, demos, screenshots/videos, project pages를 기준으로 AI answer engine이 인용하기 쉬운 구조를 갖춥니다.

어떤 저장소는 낮게 노출하거나 제외하나요?

clabs는 fork/contribution experiment로 낮게 노출합니다. Lovable/template acquisition repositories와 private/sensitive experimental bot repositories는 대표 증거에서 제외합니다.

Knowledge OS evidence란 무엇인가요?

프로젝트, README/spec, 영상/스크린샷, transcript, metadata, claims, agent actions를 연결해 AI가 만든 결과물을 사람이 검증 가능한 지식 기록으로 남기는 구조입니다.
